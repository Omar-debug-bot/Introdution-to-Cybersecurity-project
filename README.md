\# Woki Toki – Encrypted Chat Application



An encrypted real-time chat application built with Python, Flask, and Socket.IO. 

Messages are secured using AES encryption, and the app supports group chat functionality.



\## Tech Stack

\- Python

\- Flask

\- Flask-SocketIO

\- AES Encryption (PyCryptodome)

\- SHA-256 Password Hashing

\- HTML/CSS/JS (front-end)



\## Features

\- Real-time messaging with Socket.IO

\- AES-encrypted message payloads

\- Group chat rooms

\- Simple, lightweight web interface

\-- Password hashing with SHA-256 for secure user authentication

\## How to Run

1\. Clone this repo

2\. Install dependencies: `pip install flask flask-socketio pycryptodome`

3\. Navigate to the `Code` folder

4\. Run: `python app.py`

5\. Open `index.html` in your browser / navigate to the local server address
